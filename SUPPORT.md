# Support

## Where to Ask for Help

- Usage/setup questions: https://github.com/ashwingopalsamy/claude-code-theme/discussions
- Bug reports: https://github.com/ashwingopalsamy/claude-code-theme/issues
- Security issues: see `SECURITY.md`

## What to Include

For faster help, include:

- VS Code version
- extension version
- active theme variant
- reproduction steps
- screenshots (for visual issues)
