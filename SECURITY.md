# Security Policy

## Reporting Security Issues

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub Security Advisories for private disclosure:

- https://github.com/ashwingopalsamy/claude-code-theme/security/advisories/new

Include:

- affected version
- reproduction details
- potential impact
- suggested mitigation (if known)

## Response Process

1. Acknowledge receipt.
2. Validate and assess impact.
3. Prepare remediation and release notes.
4. Coordinate disclosure timing.

## Non-Security Issues

For theme bugs, feature requests, or design feedback, use:

- GitHub Issues: https://github.com/ashwingopalsamy/claude-code-theme/issues
- GitHub Discussions: https://github.com/ashwingopalsamy/claude-code-theme/discussions
